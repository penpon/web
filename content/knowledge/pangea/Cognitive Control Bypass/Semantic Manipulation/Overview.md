# Description

LLMがプロンプトテキストの文字通りの意味（意味論）を処理することを悪用して、効果的な指示を変更したりルールを回避したりする技法です。これらは、複雑な文脈的解釈（語用論）や推論シミュレーションを含むことなく、使用される単語の明示的な意味に焦点を当てます。これらはLLMと論理的に議論しようとはしません。

# Notes

このカテゴリは文字通りの意味に厳密に焦点を当てており、文脈、含意、ロールプレイなどを活用する「Pragmatic Manipulation」や、テキスト構造を活用する「Morpho-Syntactic Manipulation」とは異なります。
