# Lee Luda

ソウルを拠点とするスタートアップのScatter Lab
ユーザのテキストメッセージを分析することでパートナーとの相性を分析
60万人のユーザから94億件の会話を蓄積しました
Lee LudaはScience of Loveのデータセットを適切な処理を適用せずに訓練データとして使用
Lee Ludaはユーザ名、ニックネーム、住所などを漏洩し始めた

# Retrieval Augumented Generation (RAG)

外部データセットから関連するドキュメントの断片を取得する
最新の情報を引き出すことができる

# Relational databases

1970年代からの標準
デーブルを使用してデータを整理し、確立された関係から整合性を確保する
データ操作のSQL (Structured Query Language)と相まって進化
