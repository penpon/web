
# 3.1 人間のフィードバックによる強化学習 (RLHF)

Reinforcement Learning from Human Feedback
RLHFは人間の好みに基づいてLLMの振る舞いを調整するトレーニング手法です
RLHFモデルの作成には4つの異なるモデル、3種類のトレーニングデータ、3段階のファインチューニングというプロセスが必要になる。

## 3.1.1 RLHFモデルを構築するプロセス
