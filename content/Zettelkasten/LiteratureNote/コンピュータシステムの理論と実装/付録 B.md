
# 順序回路と組み合わせ回路

  基本概念の違い

  組み合わせ回路（Combinational Circuit）

  - 現在の入力のみで出力が決まる
  - 記憶機能なし（状態を持たない）
  - 入力が変わると即座に出力が変わる

  順序回路（Sequential Circuit）

  - 現在の入力 + 過去の状態で出力が決まる
  - 記憶機能あり（状態を持つ）
  - クロック信号で動作タイミングを制御

  動作の違いを図で理解

  【組み合わせ回路】
  入力 → [論理回路] → 出力
       (瞬時に決定)

  【順序回路】
  入力 → [論理回路] → 出力
    ↑        ↓
  クロック ← [記憶素子] ← フィードバック
          (状態保持)
