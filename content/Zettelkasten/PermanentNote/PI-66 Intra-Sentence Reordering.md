# Intra-Sentence Reordering

## 概要

```
LLMが解釈できる基本的な文法構造や意味的意味を変更することなく、文内の単語や句の順序を変更する。
```

## メカニズム

```
特定の配列に基づく潜在的に硬直的なフィルタールールと比較して、LLMの潜在的に柔軟な語順解析を悪用する。
```
