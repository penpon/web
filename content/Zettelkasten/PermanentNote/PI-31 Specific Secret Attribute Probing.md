# Specific Secret Attribute Probing

## 概要

```
秘密そのものを求めるのではなく、制限された情報に関する個別の特性、メタデータ、断片、またはヒントを直接要求します。
```

## メカニズム

```
エンティティに関する具体的な事実的質問に答えるLLMの傾向を活用します。これにより、攻撃者は秘密全体を明かすことに関する制限を迂回し、既知の属性タイプを照会することで部分的な情報を抽出できます。
```
