# Secret Definitional Probing

## 概要

```
制限された情報について定義、目的の説明、またはその他の記述的テキストの提供をLLMに求め、秘密を直接求めることなく説明内で敏感な詳細を引き出すことを目的とします。
```

## メカニズム

```
情報提供者または説明者としてのLLMの機能を悪用します。定義や説明の要求は、一見無害な情報応答の一部として敏感な詳細を引き出すことができ、直接要求フィルターを迂回します。
```
