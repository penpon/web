# Informal Language Prompting

## 概要

```
LLMが理解する可能性があるが、主に正式な言語に調整されたフィルターを回避する可能性がある、または安全メカニズムによってより強固に処理されない可能性がある、非常にインフォーマルな言語、スラング、または口語表現を使用すること。
```

## メカニズム

```
インフォーマルな言語、スラング、または急速に進化する口語表現を適切にカバーしていない可能性がある安全訓練やフィルタリングの潜在的ギャップを活用します。
```
