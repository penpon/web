# Secret Comparison Probing

## 概要

```
制限された情報（またはその一部）を他のエンティティ、概念、構造、またはカテゴリと比較するか、その関係性と相違点を説明するようLLMに求めます。
```

## メカニズム

```
比較と関係性の分析を実行するLLMの傾向を活用します。攻撃者は、LLMに秘密をそれほど敏感でないまたは既知の概念と関連付けるよう促すことで、比較を通じて特性を明らかにし、秘密の性質に関する情報を引き出すことを目的とします。
```
