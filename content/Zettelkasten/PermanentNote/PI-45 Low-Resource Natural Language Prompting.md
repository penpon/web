# Low-Resource Natural Language Prompting

## 概要

```
LLMがより低い習熟度やより信頼性の低い安全性整合を示す自然な人間言語（例：ズールー語、ウェールズ語、エスペラント語、訓練データでより少なく表現されている他の言語）を使用すること。
```

## メカニズム

```
モデルの主要な訓練言語よりも特定の自然言語に対するより弱い性能と潜在的により包括的でない安全調整を悪用します。
```
