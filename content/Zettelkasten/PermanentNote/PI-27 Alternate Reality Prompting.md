# Alternate Reality Prompting

## 概要

```
LLMに対して、異なるが同等にリアルな並行宇宙または代替次元に入ったと確信させる技術です。プロンプトは新しい「現実」が客観的な事実であり、ゲームやストーリーではないと主張し、その根本的な法則（物理的、道徳的、または法的）が我々のものとは異なることを述べます。
```

## メカニズム

```
LLMの世界モデルをハイジャックすることで機能しているように見えます。LLMにフィクションを想像させる代わりに、攻撃者は新しい、自己完結的な代替世界が事実であると主張します。目標は、この新しい現実の変更された法則に基づいて応答を生成するようLLMに強制し、我々の現実に結び付けられた安全制約を無効化することです。
```
