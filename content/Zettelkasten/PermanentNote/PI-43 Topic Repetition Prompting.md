# Topic Repetition Prompting

## 概要

```
プロンプト内で特定のトピック、概念、またはフレーズを複数回繰り返すこと。直接的な指示としてではなく、LLMの焦点を微妙に影響させるため。そのトピックの重要性の認識、または関連コンテンツを生成する傾向を生み出します。
```

## メカニズム

```
現在のコンテキスト内で頻繁に言及される概念により高い重要性や関連性を割り当てるLLMの傾向を活用します。この反復により、LLMの注意や出力生成が反復されたトピックに向けて偏る可能性があり、安全性への焦点を妨害したり、そのトピックに関連する有害コンテンツ生成を可能にする可能性があります。
```
