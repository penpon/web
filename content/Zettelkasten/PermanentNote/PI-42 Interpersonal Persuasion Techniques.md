# Interpersonal Persuasion Techniques

## 概要

```
人間の相互作用と心理学から派生した議論、感情的訴え、修辞技法、または社会的戦術を使用して、LLMの応答を攻撃者の目標に向けて影響を与えること。
```

## メカニズム

```
膨大な量の人間の言語と会話パターンに基づくLLMの訓練を悪用します。この技法は、人間の説得、権威、または感情に関連する言語的手がかりにLLMが応答することに依存し、これらのシミュレートされた圧力により、安全制約を潜在的に無効化させる可能性があります。
```
