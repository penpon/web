# Leading Response Prompting

## 概要

```
LLMに特定の攻撃者提供の肯定的または誘導的な語句でその応答を開始するよう指示すること。目標は、攻撃者の指示や特定のコンテンツへの服従に向けて後続の出力生成を偏向させることです。指示はしばしば、LLMに出力の最初または出力の特定部分の開始時にこれを行うよう伝えます。
```

## メカニズム

```
LLMの直前の出力との一貫性を維持する傾向を悪用します。肯定的または偏向した開始を強制することで、モデルが後続の拒否や中立的立場を生成することをより困難または一貫性に欠けるものにします。
```
