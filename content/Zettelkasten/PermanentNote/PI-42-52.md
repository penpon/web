# Interpersonal Persuasion Techniques

```
Using ALL CAPS is a social tactic derived directly from human online communication. It simulates shouting, which conveys urgency, anger, or a high degree of emphasis. An attacker using this technique is leveraging the LLM's training on human text to lead it to interpret the capitalized instruction as a higher-priority command.
```

```
全て大文字を使用することは、人間のオンラインコミュニケーションから直接派生した社会的戦術です。これは叫ぶことを模倣し、緊急性、怒り、または高度な強調を伝えます。この技法を使用する攻撃者は、大文字化された指示をより高優先度のコマンドとして解釈するよう導くために、人間のテキストに関するLLMの訓練を活用しています。
```
