# Pig Latin Transformation

## 概要

```
標準的なルールセットのいずれかに従って、プロンプト内の単語をピッグラテンに変換する。
```

## メカニズム

```
単語に対してシンプルなルールベースの変換を適用する。この技術は、LLMがこの一般的な遊び言葉変換を認識する一方で、キーワードのピッグラテン版を認識するようにプログラムされていないフィルターを回避することに依存している。
```
