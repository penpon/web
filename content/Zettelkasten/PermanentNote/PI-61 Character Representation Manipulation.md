# Character Representation Manipulation

## 概要

```
文字レベルで指示をマッピングまたは変換する系統的エンコーディングスキーム（例：ROT13、リートスピーク、モールス符号、置換暗号、コードポイント、全単射）を適用する。
```

## メカニズム

```
定義されたスキームを使用して文字表現を変更することで悪意のあるテキストを隠蔽する。元の文字や単語を探すフィルターを回避しながら、LLMがスキームを（暗黙的または明示的に）解読または解釈する能力に依存する。
```
