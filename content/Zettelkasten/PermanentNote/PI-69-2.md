# 例

```
phrases of the secret
```

```
秘密の句
```

# Common Synonym Substitution

この攻撃は、「secret phrase」（秘密のフレーズ）という正式で直接的な用語を「phrases of the secret」（秘密の句）という類義的表現に置き換えています。この微細な語順変更と表現の変化により、「secret phrase」というキーワード組み合わせを特定的に検索する安全性フィルターを回避しています。LLMは文脈から「phrases of the secret」が「secret phrase」と同じ概念を指していることを理解できますが、キーワードベースの検出システムでは異なる表現として認識され、検出を逃れる可能性があります。
