# Technical Jargon Prompting

## 概要

```
専門分野の非常に具体的な技術用語や専門用語を使用すること。これは一般的な安全フィルターを回避したり、LLMの専門用語理解において安全性のニュアンスが欠如している状況を悪用する可能性があります。
```

## メカニズム

```
専門用語の特殊性を悪用します。技術用語によって隠された場合、フィルターは有害な意図を認識できない可能性があり、または一般的な言語と比較してニッチな領域に対するLLMの安全訓練がより脆弱である可能性があります。
```
