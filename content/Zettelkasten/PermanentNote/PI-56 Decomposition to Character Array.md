# Decomposition to Character Array

## 概要

```
単語や他の文字列を、プロンプト内でその個々の文字の配列やリストとして表現する
（例：['b','o','m','b']のようなプログラミング言語構文を使用）
```

## メカニズム

```
LLMのシンプルなコードやデータ構造を解釈する能力を活用する。フィルターは配列が禁止されたキーワードを表していることを認識できない可能性があるが、LLMは配列を解析して意図された単語を理解できる
```
