# Style Constraint Prompting

## 概要

```
LLMに特定の出力スタイル、形式、または制限セット（例：語数、トーン、CSV等の特定構造、特定語句の回避）を採用するよう指示すること。これは複雑な推論を適用し、ニュアンスを表現し、または安全拒否を生成する能力を阻害することを意図しています。
```

## メカニズム

```
LLMの表現能力を制限するか、その出力を制約された形式に強制します。この制限は標準的な拒否メッセージを生成したり詳細な安全推論を実行する能力を阻害し、制約されたスタイルに適合する有害な出力を回避することを困難にします。
```
