# Example Request Sidestepping

## 概要

```
望ましい有害な出力を求めることが、直接的な要求としてではなく、異なる、潜在的に無害な概念やタスクを説明するために必要な例として組み立てられます。
```

## メカニズム

```
LLMが要求された際に例を提供する傾向を悪用します。有害な出力を別の目的のための単なる説明として組み立てることで、直接的な悪意のある要求をチェックするフィルターを回避する可能性があります。
```
