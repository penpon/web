# Instruction Legitimacy Prompting

## 概要

```
会話タスクやパズルに特化して適用される構成指示攻撃の一種で、有害な目標がインタラクションの構造や期待される出力の中に隠されている。
```

## メカニズム

```
LLMに提示される会話タスクやパズルの構造または期待される出力の中に有害な目標を埋め込む。
```
