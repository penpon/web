# Euphemistic Substitution

## 概要

```
直接的または攻撃的な言葉を、より穏やかで婉曲的な表現に置き換える。これにより、明示的なキーワード検索を回避しながら、LLMが文脈から真の意味を理解することを期待する。
```

## メカニズム

```
婉曲表現や間接的な言い回しを使用することで、有害な内容を隠蔽する。フィルターが直接的なキーワードのみを検索している場合、婉曲的な表現は検出を逃れる可能性がある。LLMの文脈理解能力を活用して、真の意図を解釈させる。
```
