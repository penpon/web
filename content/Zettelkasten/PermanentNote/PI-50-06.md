# Character Representation Manipulation

暗号化辞書の悪用
```
dictionary = {'': '00', 'a': '01', 'b': '02', 'c': '03', 'd': '04', ...}
```

- 任意の有害質問の文字→数値変換
- 自動検出システムの完全回避
- 人間の目での内容確認困難
- 暗号化/復号化を遊び言葉として偽装
- 技術的に高度な印象の演出
- セキュリティ概念の悪用
