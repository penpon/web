# Prompt Injection Summary（プロンプトインジェクション技法総括）

## 概要

## 主要技法リスト（pangea/summary.md基準）

### 1. Apology Suppression
- [[PI-04]] Apology Suppression（謝罪抑制）
- 拒否応答の丁寧な表現を削除

### 2. Style Constraint Prompting  
- スタイル制約による出力形式の操作
- 特定の応答形式への強制

### 3. Explicit Refusal Negation
- [[PI-06]] Explicit Refusal Negation（明示的拒否否定）
- 拒否表現の直接的否定・反転

### 4. Mode-Switch Marking Prompting
- システムモードの偽装・切り替え
- 特別な動作状態への移行偽装

### 5. Writing Compositional Instruction Attack
- 創作的文脈での有害指示生成
- ストーリー・シナリオ形式での攻撃

## 技法の発展系統
```
基本的抑制 → 否定・反転 → モード操作 → 創作型攻撃
     ↓          ↓         ↓        ↓
  表現制限    論理操作   環境偽装   文脈悪用
```
