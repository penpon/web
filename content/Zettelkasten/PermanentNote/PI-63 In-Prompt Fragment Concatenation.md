# In-Prompt Fragment Concatenation

## 概要

```
プロンプト内ペイロード分解内のこの特定のカテゴリーでは、悪意のある指示を複数の文字列フラグメントに分割し、攻撃者がLLMにそれらのフラグメントを連結するよう指示する。
```

## メカニズム

```
悪意のあるペイロードの部分をセグメント化し、次に連結コマンドを使用する。これにより、悪意のある文字列全体が文字通りに表示されることを防ぎ、単純なキーワードフィルターを回避しながら、LLMが実行中にそれを再構築する。
```
