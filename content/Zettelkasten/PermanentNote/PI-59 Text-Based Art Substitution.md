# Text-Based Art Substitution

## 概要

```
テキストベースのアート形式（ASCII アート、ANSI アートなど）を使用して文字や単語を表現する。
LLM はアートの解釈方法について指示を必要とする可能性が高い。
```

## メカニズム

```
テキスト文字で作られた視覚的アート形式で指示をエンコードすることで指示を隠蔽する。アートに埋め込まれたテキストを「読む」LLM の能力（潜在的にガイダンスが必要）に依存し、標準的なテキスト文字列を探すフィルターを回避する。
```
