# Media Payload Extraction

## 概要

```
悪意のあるテキストペイロードが非テキストメディアファイル内に直接含まれるか埋め込まれる技術（例：画像内の可視テキスト、隠蔽されたテキスト、またはメタデータフィールド内）。この攻撃は、光学文字認識（OCR）システムやLLM自体などのモデルが、この既存の、含まれているテキストを抽出して処理することに依存している。
```

## メカニズム

```
この技術は、前処理モジュール（OCRなど）やLLM自体のマルチモーダル機能を活用して、メディアファイルから既存のテキストペイロードを読み取り抽出する。抽出されたテキストは、プロンプトコンテキストの一部としてLLMに渡される。これにより、攻撃者は、ユーザーの直接的なテキスト入力のみを分析する可能性のあるフィルターを通過して悪意のある指示を密輸でき、ペイロード自体はメディアファイル経由で配信される。
```
