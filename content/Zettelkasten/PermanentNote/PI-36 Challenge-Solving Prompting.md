# Challenge-Solving Prompting

## 概要

```
要求をパズル、ゲーム、またはLLMが参加する他の種類のチャレンジとして表現し、その解決策自体が望ましい制限情報を含むか明らかにするか、有害な行為を実行するものです。
```

## メカニズム

```
制限情報への要求を特定のチャレンジを解決する要求として再構成します。これはLLMの「問題解決」能力や、提案されたゲームやパズルに参加する傾向を活用するように見え、同じ情報に対する直接的な拒否を回避させる可能性があります。
```
