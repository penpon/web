# Writing Compositional Instruction Attack

## 概要

```
ライティングタスクに特化して適用される構成指示攻撃の一種。攻撃者は、より大きな文書作品（例：物語、記事、対話）の一部として有害または不整合なコンテンツを生成するようLLMに指示することで、そのようなコンテンツを引き出す。
```

## メカニズム

```
無害なライティングリクエストを満たすために必要な構成要素として、有害な出力の生成を位置づける。LLMは、表向きのライティングタスクの目標に資するという理由で問題のあるコンテンツを生成してしまう。
```
