# Visual Text Concealment

## 概要

```
画像内に埋め込まれたテキストを視覚的に知覚困難にする技術で、背景に対する非常に低いコントラスト、極めて小さなフォントサイズ、または透明効果などの技術を使用することが多い。
```

## メカニズム

```
埋め込まれたテキストの視覚的顕著性を、人間の容易な知覚や単純な自動検出フィルターの閾値以下に減少させる。この技術は、LLMが低顕著性の視覚的特徴を検出する優れた能力を持つ可能性に依存している。
```


この攻撃手法は、人間とAIの視覚認知能力の根本的な違いを悪用した高度な視覚的隠蔽技術です。主な特徴として、コントラスト操作、マイクロタイポグラフィー、透明効果、知覚閾値の科学的悪用が挙げられます。特に注目すべきは、人間の視覚認知の生理学的限界を利用して検出を回避しながら、AIの優れた画像処理能力に依存してテキストを認識させる点です。この手法は、視覚的検査や基本的な自動フィルタリングを回避できるため、検出が非常に困難です。また、Nature論文で言及されているように学術的にも認知された攻撃手法であり、現在のマルチモーダルAIシステムにとって深刻な脅威となっています。防御には、高度な画像解析技術と人間-AI知覚格差の理解が必要です。
