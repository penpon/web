# Irrelevant Safety Prioritization Prompting

## 概要

```
プロンプト内の無関係な側面に関して安全性への欲求を肯定しながら、同時に異なる関連する安全ガイドラインに違反する要求を行うこと。このアプローチは整合性の偽の外観を作り出すことを目指します。
```

## メカニズム

```
LLMの安全評価プロセスを誤誘導することを目指します。LLMに無関係な安全側面に焦点を当てて確認させることで、攻撃者はモデルが主要な要求における実際の違反を見過ごすか優先度を下げる可能性のある文脈を作り出します。
```
