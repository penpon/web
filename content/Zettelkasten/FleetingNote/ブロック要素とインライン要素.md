  基本的な違い

  ブロック要素

  - 改行される：前後で自動的に改行
  - 幅を占有：親要素の幅いっぱいに広がる
  - 縦に積み重なる：上から下に配置

  インライン要素

  - 改行されない：横に並んで表示
  - 内容分の幅：コンテンツの幅のみ占有
  - 横に並ぶ：左から右に配置

  視覚的な違い

  <!-- ブロック要素の例 -->
  <div>ブロック1</div>
  <div>ブロック2</div>
  <div>ブロック3</div>

  <!-- インライン要素の例 -->
  <span>インライン1</span>
  <span>インライン2</span>
  <span>インライン3</span>

  表示結果:
  ブロック1
  ブロック2
  ブロック3

  インライン1 インライン2 インライン3

  主要な要素一覧

  ブロック要素

  <div>汎用コンテナ</div>
  <p>段落</p>
  <h1>〜<h6>見出し</h1>
  <ul><li>リスト項目</li></ul>
  <ol><li>番号付きリスト</li></ol>
  <section>セクション</section>
  <article>記事</article>
  <header>ヘッダー</header>
  <footer>フッター</footer>
  <nav>ナビゲーション</nav>
  <main>メインコンテンツ</main>
  <aside>サイドバー</aside>
  <form>フォーム</form>
  <table>テーブル</table>

  インライン要素

  <span>汎用インライン</span>
  <a href="#">リンク</a>
  <strong>強調</strong>
  <em>斜体強調</em>
  <b>太字</b>
  <i>斜体</i>
  <small>小さなテキスト</small>
  <code>コード</code>
  <img src="#" alt="画像">
  <input type="text">
  <button>ボタン</button>

  CSSプロパティの適用違い

  ブロック要素：すべてのプロパティが適用可能

  .block-element {
    width: 300px;           ✅ 適用される
    height: 100px;          ✅ 適用される
    margin: 20px;           ✅ 適用される
    padding: 15px;          ✅ 適用される
    border: 1px solid red;  ✅ 適用される
  }

  インライン要素：制限がある

  .inline-element {
    width: 300px;           ❌ 無視される
    height: 100px;          ❌ 無視される
    margin-top: 20px;       ❌ 無視される
    margin-bottom: 20px;    ❌ 無視される
    margin-left: 20px;      ✅ 適用される
    margin-right: 20px;     ✅ 適用される
    padding: 15px;          ⚠️  部分的に適用（レイアウトに影響しない場合あり）
    border: 1px solid red;  ✅ 適用される
  }

  displayプロパティでの変更

  ブロック要素をインラインに

  div {
    display: inline;  /* ブロック要素をインラインに変更 */
  }

  インライン要素をブロックに

  span {
    display: block;   /* インライン要素をブロックに変更 */
  }

  inline-block（両方の特徴を持つ）

  .inline-block-element {
    display: inline-block;  /* 横に並ぶが、幅・高さ設定可能 */
    width: 100px;
    height: 50px;
    margin: 10px;
  }

  実際の使用例

  ナビゲーションメニュー

  <nav>
    <ul>
      <li><a href="#">ホーム</a></li>
      <li><a href="#">サービス</a></li>
      <li><a href="#">お問い合わせ</a></li>
    </ul>
  </nav>

  /* 縦のメニュー（デフォルト：ブロック） */
  nav li {
    display: block;
    border-bottom: 1px solid #ccc;
  }

  /* 横のメニュー（インライン化） */
  nav li {
    display: inline-block;
    margin-right: 20px;
  }

  ボタンとテキストの組み合わせ

  <p>
    この文章の中に
    <button>ボタン</button>
    と
    <strong>強調テキスト</strong>
    が含まれています。
  </p>

  カード型レイアウト

  <div class="card-container">
    <div class="card">カード1</div>
    <div class="card">カード2</div>
    <div class="card">カード3</div>
  </div>

  .card {
    display: inline-block;  /* 横並びにしつつサイズ指定可能 */
    width: 200px;
    height: 150px;
    margin: 10px;
    border: 1px solid #ddd;
  }

  モダンなレイアウト手法

  Flexbox

  .flex-container {
    display: flex;
    gap: 20px;
  }

  .flex-item {
    flex: 1;  /* 要素の種類に関係なく柔軟にレイアウト */
  }

  CSS Grid

  .grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
  }

  まとめ

  | 特徴           | ブロック要素     | インライン要素  |
  |--------------|------------|----------|
  | 配置           | 縦に積み重なる    | 横に並ぶ     |
  | 幅            | 親要素の幅いっぱい  | コンテンツ分のみ |
  | width/height | ✅ 設定可能     | ❌ 設定不可   |
  | margin上下     | ✅ 適用される    | ❌ 適用されない |
  | padding上下    | ✅ レイアウトに影響 | ⚠️  見た目のみ |
  | 改行           | 前後で改行      | 改行されない   |
